HTML

  • HTML is used to build the "frame" of a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element is an example of a semantic HTML element. In semantic HTML elements, the name of the element describes its content and the role it plays on the webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • always work on a feature branch
  • It is important to frequently sync the local version of your code with the remote version in GitHub. Working on an outdated version can lead to big sync problems.
  • git pull origin main: syncs our local branch with the remote branch in GitHub
  • git add -A: adds our changes to the staging area
  • git commit -m "Enter a short descrpitive message here": commits our changes in our staging area to our repository
